# Immortal-Merge-2048 (修仙版 2048)

## 游戏简介
一款基于经典 2048 游戏改编的修仙主题合并游戏。在 5x5 的棋盘上，通过合并相同的境界方块，从「练气初期」一路修炼到「真仙之境」。

## 游戏特色
- 🎮 **5x5 网格**：比传统 2048 更大的棋盘，包含 25 个不同等级的境界
- 🧘 **修仙主题**：25 种境界名称，从练气初期到真仙之境
- 🤖 **自动运行**：一键 AI 自动游玩功能
- 🏆 **排行榜**：本地存储的历史成绩记录
- 🦊 **动态萌宠**：可爱的灵宠伙伴为你加油打气
- 📱 **响应式设计**：适配各种屏幕尺寸
- 🎨 **精美界面**：使用 Bootstrap 5 构建的现代 UI

## 境界等级
1. 练气初期 → 2. 练气中期 → 3. 练气后期
4. 筑基初期 → 5. 筑基中期 → 6. 筑基后期
7. 结丹初期 → 8. 结丹中期 → 9. 结丹后期
10. 元婴初期 → 11. 元婴中期 → 12. 元婴后期
13. 化神初期 → 14. 化神中期 → 15. 化神后期
16. 炼虚初期 → 17. 炼虚中期 → 18. 炼虚后期
19. 合体初期 → 20. 合体中期 → 21. 合体后期
22. 大乘初期 → 23. 大乘中期 → 24. 大乘后期
25. **真仙之境** (最高境界)

## 操作方法
- ⌨️ **方向键** 或 **WASD**：移动方块
- 🔄 **相同境界**：合并晋升到下一境界
- 🎯 **目标**：合成最高境界「真仙之境」
- 🤖 **自动运行**：点击按钮让 AI 帮你玩
- ⏸️ **暂停**：随时暂停游戏

## 技术栈
- HTML5
- CSS3 (自定义样式 + Bootstrap 5)
- JavaScript (ES6+)
- LocalStorage (保存分数和排行榜)

## 运行方式
直接在浏览器中打开 `index.html` 文件即可开始游戏。

## 游戏规则
1. 每次移动后会在随机空位生成一个新的低阶方块
2. 相同境界的方块相撞时会合并成更高一级的境界
3. 当棋盘填满且无法合并时，游戏结束
4. 分数会根据合并的境界等级计算
5. 游戏结束后分数会自动保存到排行榜
